# NutriPal - телеграм бот доя подсчета калорий и макроэлементов. 
NutriPal — это Telegram-бот, разработанный в рамках курсовой работы.
Он помогает пользователю получать рекомендации по питанию, учитывать продукты и формировать рацион. 
# Цель проекта 
Предоставлять рекомендации по питанию; 
Отслеживать продукты которые употребляет пользователь;
Демонстрировать основы работы с Telegram Bot API и обработкой сообщений;
